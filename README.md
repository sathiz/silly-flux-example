silly-flux-example
==================
> A silly account editor prototype for my awesome [day job](http://www.mindflash.com).

Install dependencies:
```shell
npm i; cd server; npm i; cd ..
```
Run:
```shell
node server/index.js & gulp
```
Then go to: [http://localhost:8080/](http://localhost:8080/)
Deploy:
```shell
gulp deploy
```
